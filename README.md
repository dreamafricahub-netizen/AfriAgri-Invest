# AfriAgri Invest - Prototype

Prototype de plateforme d'investissement agricole virtuel.

## Technologies
- Next.js 14+ (App Router)
- Tailwind CSS v4 (Mobile First)
- Zustand (State Management)
- Framer Motion (Animations)
- Recharts (Graphiques)

## Installation

1. Installer les dépendances :
```bash
npm install
```

2. Lancer le serveur de développement :
```bash
npm run dev
```

3. Ouvrir [http://localhost:3000](http://localhost:3000)

## Fonctionnalités Clés
- **Investissement** : Achetez des packs de fermes virtuelles.
- **Simulation** : Gains journaliers composés de 8%.
- **Persistance** : Vos données sont sauvegardées localement (LocalStorage).
- **Responsive** : Optimisé pour mobile.
